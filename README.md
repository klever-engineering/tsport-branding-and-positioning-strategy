# TSport Swim Platform — Brand & Positioning Prototype

A clickable, front‑end prototype for a premium swim booking platform. It showcases a modern brand system, strategic positioning, and end‑to‑end user flows for families, venues, and organizations.

This repo is a **demo** and is not connected to any real platform or production data. All elements are adaptable to revision and discussion.

## Highlights
- Premium hero system with trust signals and social proof.
- Stakeholder journeys for families, venues, instructors, and corporate programs.
- Strategy, resources, timeline, and offer pages for branding and positioning delivery.
- Multi‑tab venue details with certification badges, safety ratios, and reviews.
- Member dashboard focused on progress, achievements, and bookings.
- Consistent design system tokens for typography, gradients, cards, and CTAs.

## Pages
- `index.html` — Landing page + programs + how it works + pricing.
- `venue.html` — Premium venue profile + booking + reviews.
- `dashboard.html` — Logged‑in member experience.
- `marketing.html` — Branding + positioning strategy.
- `resources.html` — Resource hub and UI kit overview.
- `timeline.html` / `roadmap.html` — Delivery timeline and milestones.
- `offer.html` — Commercial offer and value breakdown.
- `legal.html` — Image credits and disclaimers.

## Local Development
No build tools required. Serve the folder with a static server:

```bash
python3 -m http.server 8080
```

Then open:

```
http://127.0.0.1:8080/index.html
```
